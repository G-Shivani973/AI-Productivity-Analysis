# AI Productivity Analysis System

## Overview
This project is an AI-based productivity analysis dashboard built using Python and Streamlit.

It analyzes employee task data, calculates productivity scores, detects burnout risk, and visualizes productivity trends.

## Features
- Bulk CSV upload support
- Automatic productivity score calculation
- Burnout risk detection
- Interactive dashboard visualization
- Scalable for large datasets

## Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run

Install dependencies:
python -m pip install pandas matplotlib streamlit

Run the app:
python -m streamlit run app.py
